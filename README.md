# Expanding On This Project

After the annoying beeps (lol), I wanted to expand more on this project. So I decided to make a Melody Player. With this, it defines musical note frequencies using constants, then stores a sequence of notes and their corresponding durations in arrays. Using a "for" loop, the Arduino iterates through each note, playing it one at a time using the tone() function. Each note is played for a specific duration, followed by a short delay to separate the notes and make the melody clear. After doing this in the Arduino IDE this is the end product.
